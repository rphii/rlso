NAME

so - create a new So string at run-time  

LIBRARY

rphii/so (librphiicso,~-lrphiicso)  

SYNOPSIS

#include <rphii/so.h>

#define so_l(s)           (So){ .ref.str = (s), .ref.len = sizeof((s)) - 1 }
 

DESCRIPTION

Create an So string from a c-string at compile-time.  

RETURN VALUE

Always creates a valid So string.  

EXAMPLES

Below shows how to create an So string and print it. 
#include <stdio.h>
#include <rphii/so.h>

int main(void) {
    So str = so("Hello, World!");
    printf("%.*s", SO_F(str));
    return 0;
}

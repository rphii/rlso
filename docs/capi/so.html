so

up: so-core

Synopsis

#include <rphiic/so.h>
#define so(str)     so_ll((str), sizeof(str) - 1)

Description

Creates one So string from a c-string.

The string will always be a reference, and not stored in stack via this method.

The length is estimated at compile time, via the sizeof(str) - 1. It needs to subtract one, because the \0 terminator.

Return Value

Always returns an So string.

Examples

#include <rphiic/so.h>
#include <stdio.h>

int main(void) {
    So str = so("Hello, World!");

    printf("%.*s\n", SO_F(str));
    return 0;
}
